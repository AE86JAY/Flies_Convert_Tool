# 多功能文件格式转换器

这是一个功能强大的文件格式转换工具集合，支持各种文件类型之间的相互转换，包括文档、电子书等多种格式。

## 功能特性

- **文档格式转换**：支持各种文档格式之间的转换
- **电子书格式转换**：支持常见电子书格式的互相转换
- **文件分割与合并**：可以将大型文件分割成多个小文件，或合并多个文件
- **补丁应用**：支持应用补丁文件修改代码或文档
- **工具函数库**：提供通用的工具函数支持

## 项目结构

```
├── .github/          # GitHub Actions工作流
├── scripts/          # 主要脚本文件
│   ├── apply_patches.sh     # 应用补丁脚本
│   ├── convert_docs.sh      # 文档转换脚本
│   ├── convert_ebooks.sh    # 电子书转换脚本
│   ├── setup.sh             # 环境设置脚本
│   ├── split_files.sh       # 文件分割脚本
│   └── utils.sh             # 工具函数脚本
├── .gitignore        # Git忽略文件
└── README.md         # 项目说明文件
```

## 使用方法

1. 首先运行设置脚本安装必要的依赖：
   ```bash
   ./scripts/setup.sh
   ```

2. 根据需要使用相应的转换脚本，例如转换文档：
   ```bash
   ./scripts/convert_docs.sh <输入文件> <输出格式>
   ```

## 系统要求

- Linux 操作系统
- Bash Shell
- 相关转换工具（会在setup.sh中自动安装）
